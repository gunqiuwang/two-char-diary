# 二字日记

每天用「两个字」和「一个爻卦」记录当天。

设计风格：[Kami](https://github.com/tw93/kami) - 暖米白底、墨蓝点缀、衬线体层次。

## 项目结构

```
miniprogram/
├── app.js                    # 小程序入口 + 六十四卦数据
├── app.json                  # 小程序配置
├── app.wxss                  # 全局样式（Kami暖米白底）
├── project.config.json       # 项目配置
├── sitemap.json              # 站点地图
└── pages/
    ├── index/                # 首页
    │   ├── index.js          # 首页逻辑
    │   ├── index.json        # 首页配置
    │   ├── index.wxml        # 首页模板
    │   └── index.wxss        # 首页样式
    └── share/                # 分享页
        ├── share.js          # 分享页逻辑
        ├── share.json        # 分享页配置
        ├── share.wxml        # 分享页模板
        └── share.wxss        # 分享页样式
```

## 设计规范（Kami风格）

| 元素 | 色值 | 说明 |
|------|------|------|
| 背景 | `#f5f4ed` | 暖米白，不用纯白 |
| 主色 | `#1B365D` | 墨蓝，唯一彩色 |
| 主文字 | `#141413` | 暖黑，带橄榄调 |
| 次文字 | `#3d3d3a` | 暖灰 |
| 辅助文字 | `#504e49` | 橄榄灰 |
| 元数据 | `#6b6a64` | 石灰色 |
| 边框 | `#e8e6dc` | 暖米色 |

## 功能特性

1. **今日二字** - 输入两个汉字记录今天
2. **爻卦生成** - 点击铜钱自动生成六爻卦象
3. **六十四卦** - 完整的六十四卦数据
4. **分享功能** - 生成精美分享卡片

## 使用方法

1. 用微信开发者工具打开 `miniprogram` 目录
2. 编译运行即可

## 设计原则

- 极简、留白、安静、克制
- 衬线体为主，层次靠字重
- 暖色调，避免冷灰
- 墨蓝点缀，克制使用
